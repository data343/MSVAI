# Cline Rules - Company Logbook

## Purpose
This logbook records key company actions, decisions, and events in chronological order to preserve institutional memory and ensure continuity.

update memory-bank/logbook.md with every started or finished task that have big impact on overall company


## Entry Structure

### Date Header Format
```
## YYYY-MM-DD
```

### Entry Format
```
### [Category] Action/Decision Title

**Who**: Key participants (names/roles)

**What**: Clear description of the action or decision

**Why**: Brief context or reasoning

**Next**: Follow-up actions with deadlines (if applicable)
```

## Categories
Use these labels to organize entries:
- **[STRATEGIC]** - Major decisions affecting company direction
- **[OPS]** - Operational changes
- **[FINANCE]** - Budget, investments, financial matters
- **[PEOPLE]** - Hiring, org changes, departures
- **[LEGAL]** - Contracts, compliance, legal issues
- **[CLIENT]** - Important client developments
- **[PRODUCT]** - Product updates or launches
- **[TECH]** - Technology implementations
